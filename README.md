# zPump

A Node.js project.

## Getting Started

### Prerequisites

- Node.js (v20.19.6 or higher)
- npm (v10.8.2 or higher)

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

### Production

```bash
pm2 start app.js
```

## Project Structure

```
zPump/

└── package.json      # Project configuration
```
